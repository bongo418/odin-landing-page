# Odin Landing Page

A landing page for The Odin Project using Flexbox for practice.

## Project Structure

- `index.html`- Landing page.
- `./css/` - Folder containing external CSS styles.
- `./images/` - Folder containing the images.
- `README.md`

## Sections

The landing page contains 4 sections.

- Hero area
- Information
- A quote
- Sign up area

## Made With

- HTML5
- Semantic tags like `<header>`, `<main>`, `<section>`, and `<footer>`
- CSS and Flexbox
